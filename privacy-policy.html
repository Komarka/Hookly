Hookly Privacy Policy

This Privacy Policy explains what Hookly accesses, how it is used, and when it is sent to third-party services.

Last updated: April 8, 2026

What Hookly Does

Hookly is a Chrome extension that helps users generate short LinkedIn outreach openers from visible profile information on the currently open LinkedIn profile page.

What Data Hookly Accesses

When activated by the user on a LinkedIn profile page, Hookly may access visible profile information such as:

  • name and headline
  • location
  • current company and education
  • about text and visible experience details
  • featured content, interests, recommendations, and visible recent activity

When Hookly Accesses Data

Hookly reads visible profile information only when the user opens the extension on a LinkedIn profile page and clicks Get replies.

How Hookly Uses Data

Hookly uses the visible profile data to:

  • generate outreach opener suggestions
  • score generated messages for how natural they sound
  • provide related message refinements such as shorter or sharper versions

Third-Party Processing

To generate and score message suggestions, Hookly may send visible profile context and generated message text to Groq through its API.

Data Storage

Hookly does not provide user accounts or operate a backend database for storing scraped LinkedIn profile data. The extension may use local browser storage for limited product behavior, such as local alpha usage tracking or extension state.

Data Selling

Hookly does not sell personal data.
